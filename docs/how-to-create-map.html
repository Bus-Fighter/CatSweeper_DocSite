How To Create Stage Map

Practical Steps

Creating Map Resource File

  1. Navigate to Engine File System, there is a StageMapInfo.
  2. Right click the folder and select "Create New/Resource", then select StageMap
  1. Select Info/StageMapInfo/StageMapInfo.tres,
  2. In the inspector Create a new item in the array, assign the newly created StageMap into the array item.

Editing Map

Project has include a simple StageMapCreator for custom map creation in editor

  1. Open the StageMapCreator scene at Scene/Editor/StageMapCreator.tscn, you will see
  1. Select the StageMapCreator node, Assign the StageMap created in Create Map Resource File into Config Data/Stage Map Data
  1. Select TileMapLayer Node, Godot TileMap Editor will show up in the bottom of the editor, draw the tile in scene in the Godot way
  1. Select StageMapCreator Node, click "Save Stage Map" Button, you will check the array elements in Floor Grids to validate, assign the Map Id and you are good to go!
  1. After entering the play mode once, you can check _Export/Info/StageMapInfo/data.json to validate the StageMapResourceInfo is parsed correctly into json file.
